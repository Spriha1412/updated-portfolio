# Portfolio Arena 🚀

Welcome to my personal frontend developer portfolio! Built with React, Vite, and Tailwind CSS, this portfolio showcases my skills, experience, and projects through an interactive, neon-themed experience.

## ✨ Features

* **Interactive Welcome Area:** A dynamic landing page with a unique parallax effect.
* **Component-Based Architecture:** Modular React components for clean, maintainable code.
* **Modern Design:** Cyberpunk/neon aesthetic with vibrant glows and smooth animations.
* **Project Showcase:** Highlights my key projects, including descriptions, tech stacks, and links to source code.
* **Responsive Layout:** fully optimized for mobile, tablet, and desktop viewing.

## 🛠️ Built With

* [React](https://reactjs.org/) - A JavaScript library for building user interfaces.
* [Vite](https://vitejs.dev/) - Next-generation frontend tooling.
* [Tailwind CSS](https://tailwindcss.com/) - A utility-first CSS framework for rapid UI development.
* [React Icons](https://react-icons.github.io/react-icons/) - High quality SVG icons for React projects.

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps:

### Prerequisites

Ensure you have Node.js and npm installed.

### Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/Spriha1412/updated-portfolio.git
   ```
2. Navigate to the project directory:
   ```sh
   cd updated-portfolio
   ```
3. Install NPM packages:
   ```sh
   npm install
   ```
4. Start the development server:
   ```sh
   npm run dev
   ```
